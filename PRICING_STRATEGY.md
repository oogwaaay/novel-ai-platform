# 定价策略更新方案

## 一、竞品定价分析

### NovelAI 定价
- **基础计划**：约 $10-15/月
- **Pro 计划**：约 $20-25/月
- **年费折扣**：约 20%
- **特点**：按上下文长度分层（4K/8K/32K tokens）

### Squibler 定价
- **个人计划**：约 $9-12/月
- **专业计划**：约 $16-19/月
- **年费折扣**：约 15-20%
- **特点**：按团队规模和协作功能分层

### 行业标准
- **Pro 计划生成次数**：通常 300-500 次/月
- **年费折扣**：15-20% 是行业标准
- **高端计划**：$30-40/月 是常见定价

## 二、成本分析

### API 成本计算
- **DeepSeek API**：约 $0.001-0.002/次生成
  - 30页小说 ≈ 2000 tokens
  - 每次生成成本：$0.001-0.002
- **Pro 计划 1000 次成本**：$1-2/月（假设用户用完）
- **Pro 计划 500 次成本**：$0.5-1/月（更安全）

### 当前定价风险
- Pro $15/月，1000次生成
- 成本占比：6-13%（风险可控但可优化）
- **建议**：降低到 500 次，提升价格到 $25/月

## 三、新定价策略

### Free 计划：体验层
- **月费**：$0
- **年费**：$0
- **生成次数**：5次/月（提升从3次）
- **最大长度**：30页
- **导出格式**：Markdown
- **功能**：基础大纲生成

### Starter 计划：工具层
- **月费**：$12/月（提升从 $8）
- **年费**：$108/年（$9/月，**25% 折扣**）
- **生成次数**：300次/月（提升从 200 次）
- **最大长度**：100页
- **导出格式**：PDF, Markdown
- **功能**：
  - 基础风格预设（3个）
  - 章节管理
  - 历史记录
  - 智能续写（上下文感知）

### Pro 计划：智能层
- **月费**：$25/月（提升从 $15，对标竞品）
- **年费**：$240/年（$20/月，**20% 折扣**）
- **生成次数**：**500次/月**（降低从 1000 次，避免成本风险）
- **最大长度**：300页
- **导出格式**：DOCX, EPUB, PDF, Markdown
- **功能**：
  - Style Memory（学习用户风格）
  - Character Management
  - AI 写作助手（段落改写、语调调整、情节建议）
  - 风格训练系统
  - 上下文优化（8000+ words）
  - 版本历史
  - 优先支持

### Unlimited 计划：协作层
- **月费**：$35/月（提升从 $20，对标竞品高端）
- **年费**：$336/年（$28/月，**20% 折扣**）
- **生成次数**：**无限**（无任何限制）
- **最大长度**：无限
- **导出格式**：所有格式
- **功能**：
  - 实时协作（多人编辑）
  - 评论系统
  - 高级版本控制（Git-like）
  - 知识库系统
  - 写作统计与分析
  - **无 API 访问**（按用户要求，不提供 API 功能）
  - 高级模板库（20+）
  - 专属支持
