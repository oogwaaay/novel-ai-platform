# 生成时间说明

## 60页小说生成时间

### 理论时间
- **60页 ≈ 15,000 字**
- **API 调用次数**：约 4-8 次（取决于优化）
- **每次 API 调用**：10-30 秒
- **总时间**：**2-4 分钟**

### 实际情况
- **最快**：1-2 分钟（网络好，API 响应快）
- **正常**：2-3 分钟
- **较慢**：3-5 分钟（网络慢或 API 繁忙）

## 不同页数的时间估算

| 页数 | 字数 | 预计时间 |
|------|------|---------|
| 60页 | 15,000字 | 2-3分钟 |
| 100页 | 25,000字 | 4-6分钟 |
| 200页 | 50,000字 | 8-12分钟 |
| 300页 | 75,000字 | 12-18分钟 |

## 优化建议

### 快速测试
1. **先用 60 页测试**：确保功能正常
2. **如果成功**：再尝试更长的
3. **如果失败**：检查 API Key 和网络

### 如果一直"生成中"

**可能原因**：
1. **网络问题**：API 响应慢
2. **API Key 无效**：导致请求失败
3. **API 余额不足**：无法完成请求
4. **代码问题**：请求卡住

**解决方法**：
1. **等待 5 分钟**：如果超过 5 分钟还没结果，可能有问题
2. **检查后端日志**：看命令行窗口是否有错误
3. **刷新页面**：重新尝试
4. **检查 API Key**：确保有效且有余额
