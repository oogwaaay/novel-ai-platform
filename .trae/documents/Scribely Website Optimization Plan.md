## 第二部分：UI/UX 核心改进（🟠）

### 2.1 专注模式布局

* **文件**：`src/components/Sidebar.tsx`

* **修改**：实现可折叠侧边栏，支持快捷键 Cmd+\\

* **技术**：Tailwind 的 `transition-all` 实现平滑动画

* **价值**：提供沉浸式写作体验，增加编辑器可用空间

### 2.2 AI 助手优化

* **文件**：`src/components/AIAssistant.tsx`

* **修改**：将固定对话框改为悬浮球 + 抽屉设计

* **技术**：`framer-motion` 实现滑入滑出动画

* **价值**：让 AI 助手不占用永久屏幕空间，按需使用

### 2.3 编辑器体验增强

* **文件**：`src/components/StoryEditor.tsx`

* **修改**：

  * 实现 Ghost Text（停止打字 3 秒后显示 AI 建议）

  * 支持 `/` 魔术指令（如 `/outline` 生成大纲，`/character` 插入角色卡）

* **技术**：使用 Tiptap 或 Slate.js 富文本编辑器

* **价值**：降低写作门槛，提升创作效率

### 2.4 积分微交互优化

* **文件**：`src/components/PointsDisplay.tsx`

* **修改**：

  * 积分变动时添加数字滚动动画

  * 扣分时数字变红，加分时变绿

  * 点击生成按钮时添加积分消耗反馈动画

* **技术**：`framer-motion` 的 `useAnimate` API

* **价值**：让积分变动更直观，减轻用户"损失厌恶"

### 2.5 深色模式与阅读模式

* **文件**：`src/components/ThemeToggle.tsx`

* **修改**：

  * 优化深色模式对比度（深灰底 #1a1a1a 配合灰白字 #e5e5e5）

  * 新增阅读模式（米黄色/羊皮纸背景）

* **价值**：提升不同场景下的写作体验

### 2.6 移动端响应式修复

* **文件**：`src/components/Sidebar.tsx`、`src/components/PointsHistory.tsx`

* **修改**：

  * 侧边栏在移动端默认隐藏，通过汉堡菜单呼出

  * 积分流水改为卡片式布局

  * 添加语音输入功能

* **价值**：支持移动端查看和灵感记录

## 技术栈

* **框架**：React 18 + Vite

* **样式**：Tailwind CSS

* **动画**：framer-motion

* **编辑器**：Tiptap 或 Slate.js

* **数据库**：Supabase

* **AI API**：DeepSeek API
