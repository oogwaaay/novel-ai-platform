# Scribely Writing Prompts

Generate a beautiful, colorized creative writing prompt from your terminal. Sci‑Fi, Mystery, and Fantasy prompts designed to spark ideas fast.

## Install

```bash
npm install -g @keeey/scribely-prompts
```

## Usage

```bash
scribely
```

This prints a single curated prompt with ANSI colors (no external dependencies).

## Why Scribely

- Instant inspiration for drafting scenes, chapters, and outlines
- Clean CLI output that works anywhere `node` runs
- Promotes focused writing sessions without setup or accounts
