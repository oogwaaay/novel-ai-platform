## Features

- **AI Novel Generator**: Generate complete novels based on genre and idea
- **AI Novel Writer**: Continue writing and expand your stories
- **Multiple Genres**: General Fiction, Mystery, Romance, Fantasy, Sci-Fi, AI-themed novels
- **Book Outlines**: Generate structured outlines for your novels
- **SEO Optimized**: Targeting keywords: novel ai, ai novel generator, ai novel writer, novels about ai

## Tech Stack

### Frontend
- React + TypeScript
- Vite
- Tailwind CSS
- React Router
- React Query
- React Quill (Rich text editor)

### Backend
- Node.js + Express
- TypeScript
- OpenAI API (GPT-4)
- JWT Authentication

## Getting Started

### Frontend
```bash
cd novel-ai-platform
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
cp .env.example .env
# Add your OPENAI_API_KEY to .env
npm run dev
```
